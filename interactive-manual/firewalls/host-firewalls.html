# Host firewall

When you send e-mail or any file from your device (computer, mobile etc) the data is divided into packets and then sent to the reciepcent. At recieving end, the packets are collected and again reaseemeble to get actual content as sent by sender. For instance, if you are living in Atlanta and you want to send gift to your uncle staying in California. Suppose your parent gave you fixed size packets and tell you to pack in this packets only. On packing you courier it. Your uncle recieves the packet. On recieving tha packet he will check if the intended reciever is him, the number of packets are right and is sender is you. If everything is right, then he will accept it.

On internet, the size of packet depends on many things like the protcol its using for transmission. Lets simulate our email-sending example.

But sometime the packet which you are recieving may have malicious content. Taking forward our previous example, if you sent 3 packets to your Uncle but your Uncle recieve 4 packets. The 4th packet is exact identical with other 3 in the sense, it contains your name as sender and uncle's name as reciever. Suppose your uncle is fastidious and he remember's that on phone you told him that you have sent 3 packets. He therefore rejects the 4 th packets and don't accepts the 4th packet from the delivery person. Exactly this is the work of firewall. The way your uncle checks for all things before accepting i.e he follows some policies for making decision on weather to accpet the packet or not, in similar way firewall also has some set of policies upon which it makes its decision.

# Software Firewall

But wait a minute? Where is a firewall on my device? I didn't see it. Don't worry! Firewall is already installed in your system by Operating System. In your system, firewall is the bit of code installed while you install Operating System. From version of Windows XP and Linux, MacOS latest version, firewalls are installed before-hand. Firewall is a software running on each individual system and is called as host level or software firewall. Host level firewall protects you from both way. It monitors incoming as well as outgoing data based on the defined set of rules. Host level firewall is mostly software firewall as it will be insane to have seprate hardware firewall for each host.

# Policies

Decision of unitended packets or which packet to not to allowed is governed by set of rules defined in the firewall. This set of rules are called 'policies' in Computer Science term. The policies can be user-defined or system-defined.

Host level firewall basic function is to

  1. Not to allow any mailicious data to enter your system
  2. Not to allow any important data to leave your system without your consent.
